Buttons.css


Responsive modern custom CSS elements

Working Example

How to Use

Add:

<script src="main.js"></script>

To the head of your html document.
Add stylesheet:

<link rel="stylesheet" type="text/css" media="screen"href="main.css"></script>

Insert buttons

<a  class="button active custom" id="buttonFFFFFF"> Test Button </a>

class options: active (switches background with text colour when hovered over), shadow (shadow when hovered over), disabled (stops button from being clicked), custom (necessary for custom button colours)

ID Options

Green (gives green button with white text),
Blue (gives blue button with white text),
Red (gives red button with white text),
Black (gives black button with white text),
Grey (gives grey button with white text),
White (gives white button with black text),
Button (followed by full length colour hex code without #. Gives a css button where shadow, active and disabled classes apply)
